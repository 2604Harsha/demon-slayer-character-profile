# Rengoku Profile Page

A simple character profile page for **Kyojuro Rengoku** from the anime _Demon Slayer: Kimetsu No Yaiba_.

## About This Project

This is a static HTML and CSS project that showcases character information, abilities, gallery images, and more related to Rengoku Kyojuro, the Flame Hashira.

**Note:**  
This project is created for educational and fan purposes only. All character rights belong to their original creators.

##  How to View

1. Clone or download this repository.
2. Open `index.html` in your web browser.

## Technologies Used

- HTML5
- CSS3 (Inline Styles)
